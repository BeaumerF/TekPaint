# TekPaint

![view](https://i.imgur.com/tdyeLMa.png)

## Dependencies
- SFML

### Load & Save
BMP and INI files (ex: res/maison)

## Keys

|    |                    |
|:--:|--------------------|
|R   |Increase red value  |
|G   |Increase green value|
|B   |Increase blue value |
|W   |Lighten the color   |
|A   |Darken the color    |
|UP  |Increase brush size |
|DOWN|Decrease brush size |
|I   |Invert colors       |

## Compile

    $ make
